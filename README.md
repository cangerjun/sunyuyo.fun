# 孙羽幽直播点歌歌单网站  
🔗 线上地址：<https://www.sunyuyo.fun>

> 本项目开箱即用，也可作为**极简网站模板**自由复用！

## 🚀 快速使用

整个网站**仅依赖一个文件**：`index.php`。  
✅ 你可以直接将它复制到你的服务器根目录即可运行。  
✅ 如果你不需要 PHP 功能（比如动态逻辑），**重命名为 `index.html` 同样能正常工作**（所有内容均为静态结构）。
✅ 如果你需要一个蜜罐的假后台入口，可以把项目中的**`login`文件夹**放入你的服务器

无需数据库、无需构建工具、零依赖 —— 真正的“复制即用”。
