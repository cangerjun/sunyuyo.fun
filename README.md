# 🎤 孙羽幽直播点歌 · 抽卡歌单网站

🔗 线上地址：<https://www.sunyuyo.fun>

> 极简 · 免费 · 开箱即用  
> 可作**个人直播歌单模板**，也可直接改成静态 HTML 使用

---

## ✨ 主要功能

- 🎵 **歌曲点播 / 抽卡**：按曲风筛选、搜索歌名/歌手、随机抽取一首歌曲
- 📺 **直播嵌入**：直接嵌入 B 站直播间，粉丝可边看直播边点歌
- 🎮 **猜数字小游戏**：内置一个简单的猜数字游戏（位于导航栏「玩个游戏」）
- 📱 **响应式设计**：手机/电脑自适应，侧边栏导航交互友好

---

## 🚀 快速使用（两种方式任选）

### 方式一：保持 PHP 版本（记录访问者）
将整个项目文件夹放入 PHP 环境（如 Apache + PHP），  
`index.php` 会自动调用 `log_visit.php` 记录访客 IP 等信息。

### 方式二：改为纯静态 HTML（推荐，无需 PHP）
**整个网站完全不依赖 PHP**，只需一步：  
将 `index.php` **重命名为 `index.html`**，即可直接运行！

```bash
# 例如在项目目录下执行
mv index.php index.html
✅ 所有功能（点歌、抽卡、猜数字、主题切换）完全正常

✅ 不再需要任何后端环境，可部署在任何静态托管服务（GitHub Pages、Vercel、Netlify 等）

✅ 删除 log_visit.php 也无影响（不再需要访问记录）

🎮 猜数字小游戏说明
入口：右侧导航栏 → 「玩个游戏」按钮

玩法：系统随机生成 1～100 之间的数字，用户输入猜测，提示「大了/小了/猜中」

特点：轻量、无依赖、纯前端实现

游戏代码位于 /game/ 目录下，可独立提取使用

🧩 模板自定义（自由修改）
你可以任意修改以下内容，没有任何强制要求：

直播嵌入的 B站房间号（修改 iframe 的 cid 参数）

右侧导航栏所有链接（GitHub、旧版、游戏、隐藏歌单等）

歌单数据（位于 js/songs-data.js，增删改歌曲条目即可）

猜数字游戏样式或难度
